# neofetch
Neofetch is a system specification display tool that displays it visually and beautifully. Built with PowerShell, Batch, and Bash, it works on any PC!

## Neofetch is tested and working on:
[X] Windows/PowerShell
[X] Windows/Batch  <sup><sub>Requires PowerShell</sub></sup>
[ ] Linux/Bash
[ ] MacOS/Bash

# To get started, first select your Operating System and your download option!
Then, run it as root or with admistrative permissions
